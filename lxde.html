Instruções para Instalar o LXDE Customizado

Siga as instruções abaixo para instalar o LXDE customizado em seu sistema.

Passo 1: Baixar o Arquivo

Abra o terminal com privilégios de root e copie e cole o seguinte comando para baixar o arquivo lxde.tar.gz:

wget https://github.com/edxlinux993/edxlinux/raw/main/lxde.tar.gz

Passo 2: Instalar o Lubuntu e o LXDM

Certifique-se de que o Lubuntu, o LXDM e o LXDE estão instalados. Use os seguintes comandos para atualizar o sistema e instalar os pacotes necessários:

apt update
apt install lxdm
apt install lubuntu

Passo 3: Descompactar o Arquivo

Vá até o diretório /usr/share e descompacte o arquivo lxde.tar.gz:

cd /usr/share
tar -vzxf lxde.tar.gz

Isso substituirá todos os arquivos e pastas relacionados ao LXDM e LXDE.

Passo 4: Limpar Arquivos Desnecessários

Após a descompactação, verifique se o arquivo lxde.tar.gz ainda está presente e, se estiver, exclua-o:

ls
rm lxde.tar.gz

Passo 5: Imagens de Personalização

Os seguintes arquivos de imagem estão disponíveis para personalização:

  • /home/kira/edxos-plus-.jpg
  • /home/kira/splash.png

Certifique-se de adicionar ou modificar essas imagens conforme desejado.
