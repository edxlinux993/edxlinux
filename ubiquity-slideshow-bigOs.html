Instruções para Instalar o Ubiquity Slideshow

Para baixar e instalar o ubiquity-slideshow-bigOs, siga as instruções abaixo:

1. Baixar o Arquivo

Clique no link abaixo para baixar o arquivo:

Baixar ubiquity-slideshow-bigOs.tar.gz

2. Instruções de Uso

Abra o terminal e siga os passos abaixo:

# Torne-se root
sudo -i

# Navegue até o diretório /usr/share
cd /usr/share

# Baixe o arquivo usando wget
wget https://github.com/edxlinux993/edxlinux/raw/main/ubiquity-slideshow-bigOs.tar.gz

# Descompacte o arquivo
tar -vzxf ubiquity-slideshow-bigOs.tar.gz

# Liste os arquivos no diretório para verificar a pasta descompactada
ls

# Você verá a pasta ubiquity-slideshow-bigOs. Para remover o arquivo compactado, use:
rm ubiquity-slideshow-bigOs.tar.gz

# Atualize o cache
sudo update-ubiquity
